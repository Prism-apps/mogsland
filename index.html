Feature

Mogs Landの3つの特長

小さなお子さまでも迷わず、保護者の方も安心して見守れる ── そんな体験を大切にしています。

直感で気持ちいい操作。

どのゲームもシンプルな操作で遊べます。説明書いらずで、お子さまが自分でルールを発見できます。最初の1回をいっしょに遊んであげるだけで、すぐひとりで楽しめます。

飽きずに遊び続けられる。

すべてのゲームにSAMPLEモードが無料で用意されています。サブスクリプションに登録すると全レベルが開放。お子さまが気に入ってから検討できます。

FAQ

よくあるご質問

保護者の方からよくいただくご質問にお答えします。

Q2歳でもひとりで遊べますか？+
基本操作はタップ・ドラッグのみなので、ひらがなが読めなくても遊べる設計です。最初の数回だけいっしょに触っていただけると、すぐ自分で楽しめるようになります。SAMPLEモードは難易度が低めに設定されています。
Q無料でどこまで遊べますか？+
全5ゲームのSAMPLEモードが無料でお楽しみいただけます。サブスクリプションに登録するとStar_1・Star_2レベルが解放され、より難易度の高い問題に挑戦できます。
Q子どもが勝手に課金してしまわないか心配です。+
サブスクリプション登録時にはペアレンタルゲート（大人向けの計算問題）を設置しています。また、AppleのFace ID / Touch ID / パスコードによる保護も働きます。iOSのスクリーンタイム設定でアプリ内課金をブロックすることも可能です。
Q広告は表示されますか？+
外部広告は一切表示しません。遊びを邪魔する広告がないため、お子さまが安心して集中して遊べます。
QiPhone・iPadどちらでも使えますか？+
はい、iPhone・iPad両方に対応しています。ご家族のデバイスでお楽しみいただけます。1つのApple IDで複数デバイスに対応しています。
Q不具合を見つけたらどうすればいいですか？+
ページ末尾のお問い合わせからご連絡ください。ご利用端末、iOSバージョン、再現手順を添えていただけると助かります。
